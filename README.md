# Helpistos (Ήλπιστος) 🤖

Ο **Helpistos** είναι ένας φορητός φωνητικός βοηθός (Voice Assistant) στα Ελληνικά, σχεδιασμένος να εκτελεί διάφορες εργασίες μέσω φωνητικών εντολών. Μπορεί να χρησιμοποιηθεί σε υπολογιστή αλλά είναι έτοιμος και για μεταφορά σε Android μέσω του Flet/Toga frameworks.

## 🚀 Λειτουργίες

- 🔊 **Μετατροπή Κειμένου σε Ομιλία (TTS):** Χρησιμοποιεί το gTTS για να "μιλάει" ελληνικά.
- 🎤 **Αναγνώριση Φωνής:** Κατανοεί ελληνικές φωνητικές εντολές μέσω της Google Speech Recognition.
- 🕒 **Ώρα και Ημερομηνία:** Σας ενημερώνει για την τρέχουσα ώρα και ημερομηνία στα ελληνικά.
- 🌦️ **Πρόγνωση Καιρού:** Παρέχει πληροφορίες για τον καιρό σε οποιαδήποτε πόλη (μέσω Open-Meteo API).
- 📰 **Ειδήσεις:** Διαβάζει τους κυριότερους τίτλους ειδήσεων από το Google News Greece.
- 📚 **Βικιπαίδεια:** Αναζητά και διαβάζει περιλήψεις από την ελληνική Wikipedia.
- ⌨️ **Αυτόματη Πληκτρολόγηση:** Μπορεί να γράψει ό,τι του λέτε σε οποιοδήποτε ενεργό πεδίο κειμένου (χρησιμοποιώντας το πρόχειρο και εξομοίωση πληκτρολογίου).

## 🛠️ Εγκατάσταση

1. **Προαπαιτούμενα:**
   - Python 3.12 ή νεότερη.
   - [uv](https://github.com/astral-sh/uv) (προτείνεται για τη διαχείριση πακέτων).

2. **Εγκατάσταση εξαρτήσεων:**
   ```bash
   uv sync
   ```

3. **Εξωτερικές βιβλιοθήκες που χρησιμοποιούνται:**
   - `SpeechRecognition`: Για την αναγνώριση φωνής.
   - `gTTS`: Για την παραγωγή ομιλίας.
   - `playsound`: Για την αναπαραγωγή των αρχείων ήχου.
   - `wikipedia`: Για την αναζήτηση πληροφοριών.
   - `requests` & `BeautifulSoup4`: Για την ανάκτηση καιρού και ειδήσεων.
   - `pyperclip` & `pynput`: Για τη λειτουργία αυτόματης πληκτρολόγησης.

## 🎤 Χρήση

Εκτελέστε το script:
```bash
python voice_assistant.py
```

Ο βοηθός θα σας πει "Έλα μου. Τι θες;" και θα αρχίσει να ακούει. Μερικές εντολές που μπορείτε να δώσετε:
- "Τι ώρα είναι;"
- "Πες μου τον καιρό" (θα σας ρωτήσει για ποια πόλη).
- "Πες μου τα νέα" ή "Πες μου τις ειδήσεις".
- "Βρες [θέμα]" (π.χ. "Βρες την Αθήνα") για αναζήτηση στη Βικιπαίδεια.
- "Γεια" για έναν απλό χαιρετισμό.
- "Τέλος", "Έξοδος" ή "Στοπ" για να τερματίσετε την εφαρμογή.

*Οτιδήποτε άλλο πείτε, ο Helpistos θα προσπαθήσει να το πληκτρολογήσει εκεί που βρίσκεται ο κέρσοράς σας.*
